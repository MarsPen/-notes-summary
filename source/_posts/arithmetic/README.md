## 算法基础理论
[指针]: ./pointer.md "pointer"

算法的根本在理解数据结构和指针，希望大家在首先理解指针之后，再认真学习下基础的数据结构，最后再理解算法。各种算法归咎根本也不过是在各种数据结构中畅游。而所有的数据结构都是由基本数据结构的变种而来。

一法通则万法通，让我们从基础学起。
